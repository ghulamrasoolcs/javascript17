Task 3:

Write a program to print a sum of even numbers in between 1-10 using a while loop.

Answer:
var a = 0;
var n = 1;

while (n <= 10) {
    if (n % 2 == 0) {
      a = a + n;
    }
    n++
  }
console.log("Sum of 1 to 10 even numbers is ", a);

Task 4:

Write a program to display a number between 1 to 5 using the do while loop.

Answer:
var i = 1;
  do {
    console.log(i)
    i++;
  } while (i <= 5);
